1. Download [choco](https://chocolatey.org)
2. Run setup.bat
3. Run tsgen.bat `SIZE` `FILES` `MERGED_FILE`
<pre>
   [SIZE]        e.g. 32x32  
                 e.g. 128  
                 e.g. 256x128  
   [FILES]       e.g. file1.png file2.png
   [MERGED_FILE] e.g. merged.png
   
   e.g. ./tsgen.bat 256x256 top.png bottom.png merged.png
</pre>
